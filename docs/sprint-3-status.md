# Sprint 3 状态

日期：2026-05-29

## 目标

Sprint 3 的目标是把复习系统从“能复习”推进到“能管理、能诊断、能持续优化”。重点是让学习者知道今天该复习什么、复习队列由哪些卡片构成、哪些内容需要重新沉淀。

## 本轮范围

- 复习队列统计
- 复习卡类型分布展示
- 后续复习队列筛选
- 后续学习项批量管理
- 后续复习效果诊断

## 已完成

### 1. 复习队列统计

- 新增 `getReviewQueueStats`
- 支持统计活跃卡片总数
- 支持统计今日到期卡片
- 支持统计新卡
- 支持统计未来卡片
- 支持统计暂停卡片
- 支持按卡片类型统计识别、听力、拼写、口语、输出卡片数量
- 复习页顶部展示卡片类型分布
- 增加复习队列统计单元测试

### 2. 复习队列筛选

- 新增 `filterReviewCards`
- 支持按全部、到期、新卡、未来筛选复习队列
- 支持按回炉队列筛选最近忘了或困难的复习卡
- 支持按识别、听力、拼写、口语、输出筛选卡片类型
- 暂停卡不进入可复习队列
- 回炉队列会根据每张卡最近一次评分动态更新
- 当前筛选无卡片时展示空状态
- 支持一键回到全部队列
- 增加复习队列筛选单元测试

### 3. 词句批量管理

- 新增批量归档词句能力
- 新增批量恢复词句能力
- 新增批量删除词句能力
- 批量归档会暂停关联复习卡
- 批量恢复会恢复关联复习卡
- 批量删除会删除关联复习卡和复习日志
- 词句本支持选择当前筛选结果
- 词句本支持清空选择
- 词句本支持批量归档、批量恢复、批量删除操作
- 增加批量词句管理单元测试

### 4. 复习结果诊断

- 新增 `summarizeReviewLogs`
- 支持统计近 7 天复习次数
- 支持统计顺利率
- 支持统计需要回炉的复习次数
- 支持统计四档评分分布
- 支持生成近 7 天每日复习趋势
- 复习页侧栏展示复习诊断
- 增加复习诊断单元测试

### 5. 复习卡详情

- 新增 `getReviewCardDetail`
- 支持关联当前复习卡和词句本条目
- 支持统计当前卡片复习次数
- 支持展示最近一次评分
- 支持标记当前卡片是否属于回炉状态
- 复习页当前卡片下方展示卡片详情
- 支持按卡片类型展示练习建议
- 回炉卡支持提示回到原材料重新输入
- 增加复习卡详情单元测试

## 验收标准

- 复习页可以看到今日到期、新卡、总卡片
- 复习页可以看到不同卡片类型数量
- 暂停卡不会计入活跃总卡片和类型分布
- 单元测试覆盖复习统计关键分支
- 复习页可以按队列状态筛选卡片
- 复习页可以按回炉队列优先练最近忘了或困难的卡片
- 复习页可以按卡片类型筛选卡片
- 筛选后评分会自动跳到当前筛选下的下一张卡
- 词句本可以批量选择当前筛选结果
- 批量归档、恢复、删除后，关联复习卡状态同步更新
- 复习页可以根据真实复习日志展示近 7 天复习诊断
- 复习页可以展示当前卡片的关联词句、来源、复习次数和最近评分
- 回炉卡可以展示对应卡片类型的学习建议和原材料入口
