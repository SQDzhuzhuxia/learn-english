# Sprint 1 状态

日期：2026-05-28

## 目标

Sprint 1 的目标是把 Sprint 0 的基础页面骨架，打磨成更接近真实每日使用的静态产品原型。重点页面：

- 今日页
- 材料库页
- 学习页
- 复习页
- 练习页
- 进度页
- 设置页

## 已完成

### 1. Mock 数据升级

- 扩展每日学习计划
- 增加今日学习队列
- 增加材料筛选项
- 增加材料进度、已知词比例、优先级、关键表达
- 增加学习材料状态、逐句内容、AI 解释
- 增加复习摘要、复习卡详情和评分选项

### 2. 今日页

- 从简单信息卡升级为每日学习驾驶舱
- 显示今日进度、30 分钟学习流程、当前材料、下一步学习队列
- 增加“继续今日学习”和“先完成复习”入口
- 强化输入驱动原则：先听读、再精学、最后跟读

### 3. 材料库页

- 增加搜索、筛选、材料分类
- 增加今日推荐材料
- 材料卡展示难度、时间、已知词比例、学习进度、关键表达
- 预留 Sprint 2 的文本导入入口

### 4. 学习页

- 增加材料进度、句子位置、剩余学习时间
- 增强当前句学习区域
- 增加慢速、循环、保存、跟读操作按钮
- 增加逐句列表、熟悉度标记
- 增加 AI 解释面板：句意、结构、易错点、可保存表达、跟读提示

### 5. 复习页

- 增加今日到期数量、新卡数量、预计复习时间
- 增强复习卡展示：英文、例句、答案、来源
- 增加四档评分：忘了、困难、一般、简单
- 增加复习队列和下一次复习提示

### 6. 练习页

- 从简单入口升级为输出能力训练台
- 增加今日推荐跟读任务
- 展示跟读步骤：听原句、慢速跟读、录音、看转写
- 增加跟读、复述、写作、场景口语四类输出训练
- 预留录音转写、AI 纠错、复习卡沉淀流程

### 7. 进度页

- 增加本周学习复盘视图
- 展示输入、输出、复习比例
- 展示每日学习分布
- 增加当前弱项分析
- 保留场景能力地图

### 8. 设置页和移动端导航

- 设置页按学习目标、AI 和语音、同步和离线分组
- 预留 API Key、服务端代理、云同步、离线能力配置入口
- 手机顶部增加进度和设置图标入口，弥补底部导航只展示 5 个主入口的问题
