# 页面原型 v0.1

日期：2026-05-27

## 1. 信息架构

第一版主导航建议包含：

- 今日
- 材料
- 学习
- 复习
- 练习
- 进度
- 设置

手机端采用底部导航，桌面端采用左侧导航。首页直接进入“今日”，不要做营销式首页。

## 2. 今日页

目标：用户打开 APP 后立刻知道今天该做什么。

主要区域：

- 今日学习模式：30 分钟 / 60 分钟
- 今日任务卡：
  - 输入材料
  - 精听/精读任务
  - 跟读或复述任务
  - 今日复习数
- 今日进度：
  - 已学习分钟数
  - 已听读句子数
  - 已保存词句数
  - 已完成复习数
- 继续学习按钮

示例任务：

```text
今日 30 分钟
1. 听读：A Visit to the Doctor，12 分钟
2. 精听：逐句循环 8 句，7 分钟
3. 跟读：录 5 个关键句，5 分钟
4. 复习：12 张词句卡，6 分钟
```

## 3. 材料库页

目标：管理所有可理解输入材料。

主要功能：

- 材料列表
- 主题筛选：故事、生活对话、美国生活、工作、自动化、移民、字幕、用户导入
- 难度筛选：太简单、合适、略难、太难
- 学习状态：未开始、学习中、已完成、需要复习
- 导入按钮：
  - 粘贴文本
  - 上传字幕
  - 上传音频
  - 输入 URL

材料卡信息：

- 标题
- 类型
- 预计学习时间
- 难度
- 已认识词比例
- 上次学习时间

## 4. 阅读/听力学习页

目标：完成大量输入和理解。

桌面布局：

- 左侧：材料段落和逐句文本
- 右侧：词句解释、保存、AI 问答
- 底部：音频播放器和逐句控制

手机布局：

- 上方：当前句子
- 中间：文本内容
- 底部：播放器和操作栏
- 解释内容用抽屉面板打开

核心操作：

- 点击单词查词
- 选中句子解释
- 逐句播放
- 循环当前句
- 慢速播放
- 隐藏/显示中文
- 保存到词句本
- 标记认识程度

AI 解释面板包含：

- 中文解释
- 简单英文解释
- 句子结构
- 常见用法
- 中文母语者容易误解的地方
- 保存为复习卡

## 5. 复习页

目标：每天复习真正快忘的词句。

复习模式：

- 看英文想中文
- 听音频选句子
- 看中文说英文
- 拼写
- 跟读

卡片操作：

- 忘了
- 困难
- 一般
- 简单

复习结束页：

- 完成数量
- 正确率
- 明天预计复习量
- 需要重新学习的材料

## 6. 练习页

练习页聚合输出能力训练。

### 跟读

- 选择来自材料的句子
- 播放原音
- 用户录音
- 语音转写
- 对比原句
- 给出句子级反馈

### 复述

- 选择一段已学材料
- 显示关键词
- 用户用英语复述
- AI 给出表达清晰度、语法、自然度反馈

### 写作

- 选择写作任务：邮件、自我介绍、工作沟通、生活消息
- 用户写英文或中文草稿
- AI 给出修正版、解释、可复习表达

### 场景口语

- 美国生活
- 工作沟通
- 自动化专业
- 移民/入籍
- 每次练习后生成 transcript 和复习卡

## 7. 进度页

目标：让用户看到自己不是在瞎学。

指标：

- 本周学习分钟数
- 本周输入分钟数
- 本周输出次数
- 阅读词数
- 听力句数
- 掌握词句数
- 复习完成率
- 连续学习天数
- 常犯错误
- 场景能力地图

场景能力地图示例：

- 看病预约：入门
- 租房沟通：未开始
- 银行业务：未开始
- 面试自我介绍：学习中
- 技术沟通：未开始
- N-400 问答：未开始

## 8. 设置页

设置项：

- 目标英语：美式英语
- 每日目标：30 分钟 / 60 分钟
- 界面语言：中文为主 / 中英混合 / 英文为主
- AI 供应商
- API Key 管理
- 语音识别方式：云端 / 本地
- 数据同步
- 数据导入导出
- 开源许可和依赖说明，后续公开发布时使用

## 9. 第一版页面优先级

P0：

- 今日页
- 材料库页
- 阅读学习页
- 复习页
- 设置页

P1：

- 跟读页
- 复述页
- 写作页
- 进度页

P2：

- 实时场景口语页
- 入籍专项页
- 工程英语专项页
- 离线下载页

## 10. 设计方向

- 工具型应用，不做营销落地页
- 中文为主，学习内容中英混合
- 页面密度适中，适合每天反复使用
- 重点突出今日任务，不让用户做太多选择
- 所有学习行为都尽量能沉淀为词句、复习卡或进度记录
