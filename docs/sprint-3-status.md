# Sprint 3 状态

日期：2026-05-29

## 目标

Sprint 3 的目标是把复习系统从“能复习”推进到“能管理、能诊断、能持续优化”。重点是让学习者知道今天该复习什么、复习队列由哪些卡片构成、哪些内容需要重新沉淀。

## 本轮范围

- 复习队列统计
- 复习卡类型分布展示
- 后续复习队列筛选
- 后续学习项批量管理
- 后续复习效果诊断

## 已完成

### 1. 复习队列统计

- 新增 `getReviewQueueStats`
- 支持统计活跃卡片总数
- 支持统计今日到期卡片
- 支持统计新卡
- 支持统计未来卡片
- 支持统计暂停卡片
- 支持按卡片类型统计识别、听力、拼写、口语、输出卡片数量
- 复习页顶部展示卡片类型分布
- 增加复习队列统计单元测试

### 2. 复习队列筛选

- 新增 `filterReviewCards`
- 支持按全部、到期、新卡、未来筛选复习队列
- 支持按识别、听力、拼写、口语、输出筛选卡片类型
- 暂停卡不进入可复习队列
- 当前筛选无卡片时展示空状态
- 支持一键回到全部队列
- 增加复习队列筛选单元测试

## 验收标准

- 复习页可以看到今日到期、新卡、总卡片
- 复习页可以看到不同卡片类型数量
- 暂停卡不会计入活跃总卡片和类型分布
- 单元测试覆盖复习统计关键分支
- 复习页可以按队列状态筛选卡片
- 复习页可以按卡片类型筛选卡片
- 筛选后评分会自动跳到当前筛选下的下一张卡
